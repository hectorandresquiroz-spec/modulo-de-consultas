Nuevo Servicio
🗄️ Configurar base de datos compartida
¿Por qué Supabase? Es gratuito, de código abierto, tus datos son privados, no requiere TI y funciona en 5 minutos. Solo tú (y quien tú invites) tiene acceso a los datos.
1

Crea tu cuenta gratuita

Ve a supabase.com → Start for free → regístrate con tu correo institucional (o Google). Es completamente gratis para equipos pequeños.

2

Crea un proyecto nuevo

Clic en New project → ponle un nombre (ej: directorio-lae) → elige una contraseña → selecciona región South America (São Paulo) → clic en Create new project. Espera ~2 minutos.

3

Crea la tabla de datos

En el menú izquierdo ve a SQL Editor → clic en New query → pega este código y presiona Run:

create table servicios (
  id bigint generated always as identity primary key,
  nombre text not null,
  tipo text default 'gobierno',
  estado text default 'activo',
  direccion text,
  telefono text,
  email text,
  encargado text,
  created_at timestamptz default now()
);

-- Permitir acceso público (ajusta según tu seguridad)
alter table servicios enable row level security;
create policy "acceso_equipo" on servicios
  for all using (true) with check (true);
4

Copia tus credenciales

Ve a Project Settings → API y copia:

PROJECT URL
ANON PUBLIC KEY (clave pública)
5

Conectar y compartir

Haz clic en conectar. Luego comparte este archivo HTML con el equipo por correo o Teams — todos quedarán conectados a la misma base de datos automáticamente.

Privacidad: Supabase almacena los datos en servidores en São Paulo (Brasil), bajo estándares ISO 27001 y SOC 2. La clave "anon" que usas es pública por diseño — solo permite acceso a tu tabla específica. Puedes agregar contraseña o autenticación en cualquier momento desde el panel de Supabase.
